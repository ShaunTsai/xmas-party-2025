# ⚠️ 重要：更新 Google Apps Script

## 為什麼需要更新？

原本的 Apps Script 使用 POST 方法，但因為 CORS 限制，前端無法正確接收回應。
新版本使用 GET 方法，可以正常運作。

## 更新步驟（5分鐘）：

### 1. 打開你的 Google Sheet
前往你的 RSVP 追蹤表單

### 2. 打開 Apps Script
點擊：**擴充功能 → Apps Script**

### 3. 替換程式碼
1. 刪除編輯器中的所有現有程式碼
2. 打開 `google-apps-script.js` 檔案
3. 複製**全部**程式碼
4. 貼到 Apps Script 編輯器
5. 點擊 **💾 儲存**

### 4. 重新部署
1. 點擊 **部署 → 管理部署**
2. 點擊現有部署旁的 **✏️ 編輯**
3. 點擊右上角的 **版本** 下拉選單
4. 選擇 **新版本**
5. 點擊 **部署**
6. 完成！（URL 保持不變）

## 測試

1. 重新整理你的網站：http://localhost:3000
2. 填寫 RSVP 表單
3. 提交
4. 檢查 Google Sheet - 應該會看到新的一行！

## 新功能

✅ 修正：表單提交現在會正確儲存到 Google Sheet
✅ 新增：中文確認信
✅ 改進：更好的錯誤處理
✅ 支援：GET 和 POST 兩種方法

## 如果還是不行？

1. 確認 Apps Script 部署設定：
   - 執行身分：**我**
   - 存取權限：**任何人**

2. 檢查瀏覽器 Console（F12）是否有錯誤

3. 在 Apps Script 中執行 `testScript()` 函數測試
